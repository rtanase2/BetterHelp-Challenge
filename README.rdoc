== Description

Survey app built for BetterHelp's technical challenge. Allows user take a survey and display the most common answer per gender as well as allowing editting of questions and answers.

== Features

* Adding questions
* Deleting questions
* Editing questions
* Adding answers
* Deleting answers
* Editing answers
* Displaying the questions and answers in a survey
* Recording a submission of the survey
* Showing statistics of submissions
